필요한 기능 : 

//게임 시스템 관련
1. 오른쪽으로 쭉 이동하는 함수
 - 트리거에 의해 중지 될 수 있어야 함. -> bool 변수를 하나 만들어서 걸을 지 정지할지를 결정해야 함

2. 트리거를 클릭하면 음악 재생 및 노트가 떨어지는 기능 구현
 - 피아노 음 1초 재생clamp를 사용해서 노트가 화면 상단에서 중단으로 떨어지도록 할 것
3. 캐릭터가 트리거를 빠져 HP가 감소하는 상황이 되면 다음 트리거로 대상 이동하는 기능 구현
 - 트리거 생성을 순차대로 하여 다음 오브젝트를 멀티플 카메라 오브젝트에 놓을 것 이전 오브젝트는 이 후 삭제
4. 캐릭터가 트리거에 빠지기 전에 노트가 무슨 음인지 제대로 맞출 경우 대상 트리거를 통과하는 기능 구현
-> 트리거에 bool 변수 하나 만들어 놓고 그것에 따라 2가지 분기를 만들어 놓을 것
5. 노트를 맞출 때 틀리면 점수 지급 없음
6. 노트를 맞추면 나비가 날아가는 효과 오브젝트생성
7. 맵 별로 트리거들 맵핑해 놓을 것
8. 카메라는 2D로 각자의 가장 가까이 있는 트리거 까지만 ZoomToFit 할 것

//캐릭터 에니메이션 관련
9. 몬스터에 부딛히면 넘어지는 모션
10. 땅굴에 빠지면 땅굴에 빠졌다가 튕겨 올라오는 모션
11. 땅굴에 빠지기 전에 bool 변수가 true면 뛰어서 지나가는 모션
12. 몬스터를 잡을 때 모션은 피리 부는 모션

//환경 관련
13. 백그라운드 오브젝트들은 루프 형태로 반복 될 것
14. 트랙 즉 캐릭터와 크리쳐들의 발판이 될 맵은 게임이 시작 할 때 로딩한 후 불러들이는 방법을 사용 할 것
